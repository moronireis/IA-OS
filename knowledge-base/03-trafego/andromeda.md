## 1. O Que e a Andromeda

### Definicao

A Andromeda e o novo sistema de inteligencia artificial do Meta Ads, resultado de uma **parceria entre Meta e Nvidia**. Combina:

- **Hardware:** Predios inteiros cheios de computadores Nvidia (capacidade de processamento massiva)
- **Software:** Inteligencia artificial + machine learning para entender a jornada de cada usuario individualmente

### O Que a Andromeda Faz

A Andromeda tem dois entendimentos simultaneos:

1. **Jornada do usuario:** O que cada usuario especificamente faz dentro da plataforma, qual caminho ele percorre ate a compra
2. **Inventario de criativos:** O que voce tem dentro das campanhas que possa ser encaixado na jornada de cada usuario

O sistema funciona como se a Meta tivesse um **closer pessoal** (vendedor) para cada usuario da plataforma, entendendo qual sequencia de anuncios mostrar para levar aquele individuo especifico a conversao.

## 2. Antes vs Depois da Andromeda

### Comparativo Fundamental

| Aspecto | ANTES (Clusters) | DEPOIS (Andromeda) |
|---------|-------------------|---------------------|
| **Estrategia** | Encontrar 1 anuncio vencedor | Ter um portfolio de anuncios vencedores |
| **Segmentacao** | Grupo de pessoas (clusters) | Personalizacao por usuario individual |
| **Criativos** | Variações do mesmo (trocar hook, body) | Criativos realmente diferentes entre si |
| **Escala** | Duplicar clusters do vencedor | Alimentar portfolio + escalar orcamento |
| **Otimizacao** | Isolar e testar 1 a 1 | Deixar o algoritmo distribuir e sequenciar |
| **Funil** | Manual (campanha topo, meio, fundo) | Automatico (Meta cria jornada por usuario) |

### O Que Mudou na Pratica

**Antes:**
1. Subir varios anuncios
2. Isolar cada um para medir desempenho
3. Identificar o "Michael Jordan" (anuncio que carrega o jogo)
4. Concentrar verba nele
5. Duplicar em clusters
6. Fazer Frankenstein (trocar hook, body entre criativos)

**Depois (Andromeda):**
1. Criar portfolio diversificado de criativos (diferentes comunicacoes)
2. Subir todos no mesmo conjunto com CBO/Advantage
3. Deixar o Meta decidir qual sequencia mostrar para cada usuario
4. O Meta monta um "quebra-cabeca" personalizado para cada pessoa
5. Machine learning entende qual jornada converte melhor por individuo

### A Nova Logica

O Meta agora mostra anuncios como um **quebra-cabeca personalizado**:

```
Usuario A: Oferta Direta → Quebra de Objecao → Por Dentro do Metodo → COMPRA
Usuario B: Quebra de Objecao → Oferta Direta → Resultados Reais → COMPRA
Usuario C: Insight Pratico → Desbloqueio Mental → Agora ou Nunca → COMPRA
```

Cada usuario recebe uma sequencia diferente, em tempo real, baseada no comportamento individual.

## 3. Os 9 Tipos de Criativos Obrigatorios

A Andromeda exige diversidade de comunicacao. Tenha criativos que cubram diferentes estagios do funil e diferentes gatilhos:

### Lista Completa

| # | Tipo | Funcao | Nivel de Funil |
|---|------|--------|----------------|
| 1 | **Hook Impulsavel** | Quebra de padrao para parar o scroll (stop thumbing) | Topo |
| 2 | **Dor / Incomodo** | Toca no problema/dor do avatar. Tira o sono da pessoa | Topo/Meio |
| 3 | **Insight Pratico** | Ensina algo util + CTA no final. Conteudo de valor | Topo |
| 4 | **Oferta Direta** | Venda agressiva para quem esta pronto para comprar | Fundo |
| 5 | **Escolha Inteligente (Smart Choice)** | Comparacao entre seu metodo e outros do mercado | Meio/Fundo |
| 6 | **Por Dentro do Metodo (Inside Method)** | Mostra a plataforma, aulas, comunidade, suporte | Meio |
| 7 | **Resultados Reais** | Depoimentos e cases de sucesso de alunos | Meio/Fundo |
| 8 | **Desbloqueio Mental** | Quebra objecoes emocionais (falta de tempo, dinheiro, capacidade) | Meio |
| 9 | **Agora ou Nunca (Now or Never)** | Urgencia e escassez real (vagas, desconto, prazo) | Fundo |

### Detalhamento dos Tipos

**1. Hook Impulsavel**
- Objetivo: Fazer a pessoa parar de rolar o feed
- Exemplos: Pessoa pulando de paraquedas, situacao inesperada, movimento visual forte
- O "stop thumbing" - primeira coisa que a pessoa ve

**2. Dor / Incomodo**
- Trabalha um problema real do avatar
- Algo que tira o sono, gera desconforto
- Faz a pessoa se identificar: "isso acontece comigo!"

**3. Insight Pratico**
- Entrega conteudo de valor real
- Ensina algo pratico e util
- No final, faz um CTA suave
- Funcao principal: pescar publico frio e alimentar funil

**4. Oferta Direta**
- Para quem ja esta pronto para comprar
- Apresenta o produto, preco, beneficios
- CTA claro e direto

**5. Escolha Inteligente**
- Comparacao: "Nosso metodo entrega A, B, C, D. Outros entregam so A e B"
- Essencial em mercados maduros com muita concorrencia
- Ajuda o lead a tomar decisao racional

**6. Por Dentro do Metodo**
- Mostra a plataforma (Hotmart, area de membros)
- Detalha aulas, modulos, comunidade, suporte
- "Compra sem ver" diminui - mostra o que vao receber

**7. Resultados Reais**
- Depoimentos de alunos que tiveram resultado
- Focar em pessoas que comecaram do zero
- Quebra a objecao: "Sera que funciona pra mim?"

**8. Desbloqueio Mental**
- Toca no emocional
- Quebra travas: falta de dinheiro, tempo, autoconfianca
- Diferente de "dor" - aqui e sobre o interno, o emocional

**9. Agora ou Nunca**
- Urgencia real: "vagas limitadas", "desconto acaba hoje"
- Ultimo empurrao para quem ja passou por todo o funil
- Deve ser verdadeiro (escassez real)

## 4. Mix Obrigatorio: Estaticos + Videos

### Por Que Ambos

A Andromeda precisa de **variedade de formatos** para testar diferentes combinacoes por usuario:

- Alguns usuarios respondem melhor a **video** (demonstracao, emocao, storytelling)
- Outros respondem melhor a **estaticos** (imagem, carrossel, info rapida)
- O algoritmo descobre qual formato funciona para cada pessoa

### Quantidade Recomendada

| Recomendacao | Quantidade |
|-------------|-----------|
| **Meta oficial** | 10 a 50 criativos por conjunto |
| **Teste pratico (Tabari)** | 5 a 15 criativos e ja funciona bem |
| **Minimo viavel** | 5 criativos com comunicacoes diferentes |

> **Importante:** Nao e sobre quantidade pura, e sobre **diversidade de comunicacao**. 10 criativos com o mesmo hook e o mesmo corpo NAO e o que a Andromeda quer.

## 5. Otimizacao na Era Andromeda

### Regra Critica: Nao Pausar Alto-Volume com Baixo ROAS

**Cenario comum pos-Andromeda:**
- Criativo "Agora ou Nunca" tem ROAS alto e recebe a maioria das vendas
- Criativo "Insight Pratico" gasta bastante mas tem ROAS baixo ou zero vendas

**Reacao errada:** Pausar o "Insight Pratico" porque nao vende

**Entendimento correto:**
- O "Insight Pratico" esta **pescando publico frio** e alimentando o funil
- Ele pega pessoas de topo que ainda nao estao prontas para comprar
- Depois, essas pessoas recebem outros criativos e convertem no "Agora ou Nunca"
- Pausar o "Insight Pratico" = cortar o topo do funil = vendas caem depois

### O Que Pausar (Desativar)

Pausar criativos que:
- **Nao estao recebendo verba** (gasto muito baixo comparado aos outros: R$10, R$15 enquanto outros gastam R$100-500+)
- Isso indica que o Meta ja testou e concluiu que esse criativo nao e bom
- As pessoas estao pulando, nao engajam

### Resumo da Logica de Otimizacao

| Situacao | Acao |
|----------|------|
| Criativo gastando bem + ROAS bom | Manter e escalar |
| Criativo gastando bem + ROAS baixo | **Manter** (pode estar alimentando funil) |
| Criativo gastando pouco + ROAS bom | Manter (bonus) |
| Criativo gastando quase nada | **Pausar** (Meta ja rejeitou) |

## 6. Campanhas de Teste vs Campanhas de Escala

### Estrutura Recomendada

```
CONTA DE ANUNCIOS
├── CAMPANHA DE ESCALA (principal)
│   └── Conjunto Advantage (CBO)
│       ├── Criativos validados (os que performam)
│       └── Todos os 9 tipos presentes
│
└── CAMPANHA DE TESTE (separada)
    └── Conjunto Advantage (CBO)
        ├── Novos criativos sendo testados
        └── Isolado para nao afetar escala
```

### Fluxo de Criativos

1. **Criar novos criativos** e subir na campanha de teste
2. **Testar isoladamente** - nao misturar com campanha que esta vendendo
3. **Se performar bem** (conversoes, bom gasto) → promover para campanha de escala (Serie A)
4. **Se nao performar** → continuar testando na campanha separada (Serie B)
5. **Se performar muito mal** → desativar (Serie C)

### Por Que Separar

Se voce coloca criativo novo direto na campanha de escala:
- O Meta vai tirar verba dos criativos que estao convertendo para gastar no novo
- Se o novo nao performar, puxa o ROAS da campanha inteira para baixo
- Voce nao sabe a performance real do novo criativo

---

## 7. Escala na Era Andromeda

### Regra dos 20-30%

- Aumentar orcamento **20-30% por dia** no maximo
- Monitorar ROAS e CPL apos cada aumento
- Se metricas se mantiverem, continuar escalando
- Se piorarem, voltar ao orcamento anterior e aguardar 2-3 dias

### Campanha Advantage (Setup)

A campanha Andromeda e extremamente simples:

1. **Tipo de campanha:** Advantage (CBO)
2. **Publico:** Aberto (Advantage+ Audience)
3. **Posicionamentos:** Advantage+ (automatico)
4. **Criativos:** Portfolio diversificado (9 tipos)
5. **Configuracao de publicidade:** Definir publico engajado e cliente existente

> "A parte de subir a campanha em si com a Andromeda e muito facil. E uma campanha toda no Advantage, CBO, publico aberto, posicionamentos automaticos. O segredo esta nos criativos e na otimizacao." - Leonardo Tabari

## 8. API de Conversoes - Importancia Critica

### Por Que a API e Mais Importante que Antes

Com a Andromeda mostrando multiplos criativos em sequencia para cada usuario:
- A **atribuicao de vendas** fica mais complexa
- Sem API, o Meta pode atribuir a venda ao criativo errado
- Voce pode achar que um criativo esta vendendo quando na verdade e outro
- Decisoes de otimizacao ficam erradas

### Com API Configurada

- Retorno de dados mais preciso para o gerenciador
- Atribuicao correta de qual criativo realmente converteu
- Dados mais confiaveis para decisao de otimizar/pausar/escalar

### Sem API

- Atribuicao "bagunçada" entre criativos
- Primeiro ou ultimo criativo visto pode receber credito indevido
- Risco de tomar decisoes erradas na otimizacao

## 9. Transicao: O Que Fazer Agora

### Se Suas Campanhas Estao Vendendo Bem (Modelo Antigo)

1. **Nao pause!** Continue extraindo o maximo
2. Comece a testar o modelo Andromeda em campanha separada
3. Crie criativos diversificados (9 tipos)
4. Monitore resultados paralelos
5. Quando campanha antiga parar (toda campanha para), ja tenha a nova rodando

### Se Suas Campanhas Nao Estao Performando

1. Comece a testar o modelo Andromeda imediatamente
2. Sua conta/BM pode ja estar atualizada para o novo algoritmo
3. Monte portfolio de criativos diferentes
4. Suba campanha Advantage com publico aberto
5. Pode ser exatamente o que faltava para mudar seus resultados

### Mudanca de Mindset para Producao de Criativos

| PARE de fazer | COMECE a fazer |
|---------------|----------------|
| So trocar hooks entre criativos | Criar criativos com comunicacoes completamente diferentes |
| Buscar 1 unico anuncio vencedor | Construir portfolio diversificado |
| Duplicar clusters do vencedor | Alimentar campanha com criativos de diferentes funcoes |
| Fazer 5 versoes do mesmo ad | Fazer 1 de cada tipo (9 tipos) |

## 10. Resumo Pratico - Checklist Andromeda

- [ ] Configurar API de Conversoes na conta
- [ ] Definir publico engajado e cliente existente nas configuracoes de publicidade
- [ ] Criar pelo menos 5-9 criativos com comunicacoes diferentes
- [ ] Incluir mix de estaticos + videos
- [ ] Criar campanha Advantage (CBO) com publico aberto
- [ ] Criar campanha de teste separada para novos criativos
- [ ] Nao pausar criativos com gasto alto e ROAS baixo (podem estar alimentando funil)
- [ ] Pausar apenas criativos que nao recebem verba
- [ ] Escalar 20-30% por dia
- [ ] Promover criativos validados da campanha de teste para a campanha de escala
