# Skill /trafego - Planejamento de Campanha de Trafego

Quando o usuario invocar `/trafego`, siga este processo:

## 1. Coleta de Informacoes

Pergunte:
- Plataforma (Meta Ads, Google Ads, etc.)
- Objetivo (leads, vendas, awareness, remarketing)
- Orcamento diario/mensal
- Nicho e produto
- Pixel/API ja configurados?
- Business Manager existe?
- Audiencia atual e temperatura
- Historico de campanhas

## 2. Consultar Knowledge Base

Leia os arquivos de `knowledge-base/03-trafego/`:
- `sobral-principios.md` e `sobral-meta-ads.md` para fundamentos e estrutura
- `andromeda.md` para estrategia de criativos pos-Andromeda
- `cpl-otimizacao.md` para metricas e otimizacao
- `rastreamento-utms.md` para setup de tracking

## 3. Gerar Plano de Campanha

Entregue:
- **Setup checklist**: Pixel, API, BM, UTMs
- **Estrutura de campanhas**: naming convention Sobral, CBO vs ABO, fases
- **Audiencias por temperatura**: fria/morna/quente com tamanhos estimados
- **Briefing de criativos**: 9 tipos Andromeda com descricao para cada
- **Orcamento por fase**: teste (CPM x 15) → validacao → escala (20-30%/dia)
- **Metricas-alvo**: CPL, CTR, CPM, ROAS por fase
- **Calendario de otimizacao**: intervalos de revisao (14/30/60/90 dias)
- **Plano de escala**: incremento diario, quando pausar, quando criar novos conjuntos
