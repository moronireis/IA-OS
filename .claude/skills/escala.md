# Skill /escala - Diagnostico e Plano de Escala

Quando o usuario invocar `/escala`, siga este processo:

## 1. Coleta de Informacoes

Pergunte:
- Faturamento atual (mensal/anual)
- Objetivo de faturamento
- Modelo de negocio (lancamento, perpetuo, high ticket, hibrido)
- Tamanho da equipe atual
- Principais metricas: ROAS, CPL, taxa conversao, ticket medio
- Gargalos percebidos (trafego? conversao? entrega? equipe?)
- Ferramentas/automacoes atuais

## 2. Consultar Knowledge Base

Leia arquivos de TODOS os dominios:
- `01-lancamentos/ciclos-evolutivos.md` para progressao de faixas
- `01-lancamentos/lancamento-pago-baldan.md` para gestao de equipe (14-15 ideal)
- `02-high-ticket/delegacao-mentores.md` para 5 fases de delegacao
- `03-trafego/andromeda.md` para escala de trafego (20-30%/dia)
- `05-ia-automacao/` para automacoes com IA

## 3. Diagnostico de Gargalos

Analise as 5 areas:

| Area | Sintomas | Solucao |
|------|----------|---------|
| **Trafego** | CPL alto, volume baixo | Otimizar criativos, diversificar audiencias |
| **Conversao** | Muitos leads, poucas vendas | Revisar copy, oferta, objecoes |
| **Produto** | Vendas OK, reclamacoes altas | Melhorar entrega, criar AR |
| **Equipe** | Sobrecarregado, sem delegar | 5 fases FHT, max 14-15 pessoas |
| **Financeiro** | Receita alta, margem baixa | Revisar custos, otimizar ROAS |

## 4. Gerar Plano de Escala

Entregue:
- **Diagnostico** dos gargalos principais (top 3)
- **Plano de escala por fase** (30/60/90 dias)
- **Delegacao necessaria**: quais funcoes terceirizar/contratar (5 fases FHT)
- **Automacoes recomendadas**: IA para calls, agentes ChatGPT, tracking automatico
- **Projecao financeira**: baseada em benchmarks reais
- **Progressao de faixa**: onde esta e proximo marco (semente→6em7→marrom→preta)
- **KPIs para acompanhar**: metricas criticas com frequencia de revisao

## 5. Benchmarks de Escala

- Equipe ideal: 14-15 pessoas (Baldan)
- Escala trafego: 20-30%/dia (Andromeda)
- Participacao comercial: 83-86% (Baldan)
- Ciclos para 6em7: ~7 em media
- Espacamento: 45-60 dias entre lancamentos
- Faixa Marrom: R$300K+/lancamento
- Faixa Preta: R$2M+/ano
