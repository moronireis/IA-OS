# Skill /lancamento - Planejamento de Lancamento Digital

Quando o usuario invocar `/lancamento`, siga este processo:

## 1. Coleta de Informacoes

Pergunte ao usuario (pode adaptar conforme contexto):
- Qual o nicho e subnicho?
- Quem e o avatar (cliente ideal)?
- Qual a Roma (promessa de transformacao)?
- Ja fez algum lancamento antes? Qual tipo e resultados?
- Tamanho da audiencia atual (seguidores, lista email, WhatsApp)
- Orcamento disponivel para trafego
- Tem equipe? Quantas pessoas?
- Qual o objetivo financeiro?

## 2. Consultar Knowledge Base

Leia os arquivos relevantes de `knowledge-base/01-lancamentos/`:
- `fl-formula-lancamento.md` para visao geral e decisao de tipo
- O arquivo especifico do tipo escolhido (semente, classico, pago-baldan, pago-tabari, projeto-black)
- `ciclos-evolutivos.md` para progressao
- `nicho-avatar-roma.md` para validar fundamentos

## 3. Decidir Tipo de Lancamento

| Situacao | Tipo Recomendado |
|----------|-----------------|
| Nunca lancou, sem produto | Semente |
| Ja vendeu no semente, tem produto | Classico |
| Quer escalar com equipe enxuta | Pago (Baldan) |
| Quer lancamentos semanais | Pago Turbo (Tabari) |
| Base de clientes existente | Black (combo/bundle) |

## 4. Gerar Plano

Entregue ao usuario:
- **Tipo de lancamento recomendado** com justificativa
- **Timeline completa** com datas reais (baseado em ciclos de 45-60 dias)
- **Checklist de preparacao** dividido em fases (pre/durante/pos)
- **Estrutura de CPLs** (se aplicavel)
- **Orcamento estimado** de trafego
- **Metricas-alvo**: ROAS 1.0-1.2 (pago), unique views 70%+, conversao 7-10%+
- **Proximos 5 passos imediatos**

## 5. Benchmarks de Referencia
- Media de 7 ciclos para atingir 6em7 (R$100K/7 dias)
- Espacamento ideal: 45-60 dias entre lancamentos
- Baldan Black Friday: R$130K → R$2.5M (ROAS ~19x)
- Tabari: R$281K/7 dias, R$1M/30 dias
- Frequencia conteudo: 2 Raiz + 7 Nutella/semana
