# Skill /copy - Geracao de Copy para Lancamentos

Quando o usuario invocar `/copy`, siga este processo:

## 1. Coleta Obrigatoria (ANTES de escrever)

Pergunte:
- Tipo de copy: CPL1, CPL2, CPL3, criativo (estatico/video), landing page, pitch, email, VSL
- Nicho e avatar
- Roma (promessa de transformacao)
- Produto/servico
- Tom de voz (formal, casual, urgente, inspiracional)
- Objecoes principais do avatar

## 2. Consultar Knowledge Base

Leia de `knowledge-base/04-conteudo-copy/`:
- `scripts-cpl.md` para CPLs e Video de Vendas
- `copy-criativos.md` para criativos e landing pages
- `pitch-alta-conversao.md` para pitches
- `reels-short-form.md` para short-form

## 3. Frameworks por Tipo

**CPL1**: Apresentacao → Prova → Historia → Conteudo → Oferta → Fechamento
**CPL2**: Roma → Prova → Review → Conteudo → Oferta → Fechamento
**CPL3**: Q&A / Amostra / Indice-Mapa
**VSL**: Promessa → Prova → Historia → Oferta
**Criativos**: 5 angulos de hook (contra-intuitivo, paradoxal, direto, social proof, urgencia)
**Landing Page**: 11 secoes (headline, sub, prova, dor, oportunidade, para quem, FAQ, preco, autoridade, CTA, garantia)
**Pitch**: Avatar mapping (4 dimensoes) → falsas crencas → blocos do pitch

## 4. Output

Entregue:
- **Copy completa** pronta para uso
- **2-3 variantes A/B** (headlines, hooks, CTAs)
- **Character counts** (para formatos com limite)
- **Notas de producao** (indicacoes visuais se video)
- **Objecoes endereçadas** por trecho

Principio central: ALINHAMENTO, nao persuasao.
