# Skill /high-ticket - Sistema de Vendas High Ticket

Quando o usuario invocar `/high-ticket`, siga este processo:

## 1. Coleta de Informacoes

Pergunte:
- Qual o nicho e avatar?
- Qual a Roma (transformacao prometida)?
- Faixa de preco desejada (R$5K-100K+)?
- Ja faz sessoes 1a1? Quantas? Taxa de conversao?
- Tem equipe comercial? Quantos closers?
- Como gera leads hoje? (SLG, trafego, indicacao)
- Principais objecoes que escuta?

## 2. Consultar Knowledge Base

Leia os arquivos de `knowledge-base/02-high-ticket/`:
- `fht-masterplan.md` para visao completa do sistema
- `sessao-1a1.md` para framework de sessao
- `arvore-objecoes.md` para objecoes + 4 E's
- `full-sales-system.md` para trilhas de vendas
- `delegacao-mentores.md` se tiver/quiser equipe

## 3. Gerar Entregaveis

Produza:
- **Estrutura da oferta**: pacotes DFY/DWY/DIY com pricing e ancoragem
- **Arvore de objecoes personalizada**: top 5 objecoes do nicho com respostas em 3 camadas
- **Script de sessao 1a1**: abertura → diagnostico (SPIN) → oferta (4 E's) → fechamento
- **Plano de SLG**: mensagens de desarme e conducao para Instagram
- **Funil de aula semanal**: estrutura de captacao + nurturing
- **Plano de delegacao** (se aplicavel): 5 fases, guia do mentor
- **Metricas-alvo**: taxa conversao, ticket medio, show-up rate, CAC
