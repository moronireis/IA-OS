# Base de Conhecimento IA-OS

## O que é a Base de Conhecimento

A Base de Conhecimento do IA-OS é um acervo de **28 documentos estruturados** organizados em **5 domínios** de marketing digital. Todo o material foi curado e sintetizado a partir de aproximadamente **~500MB de conteúdo bruto** (vídeos, PDFs, lives, mentorias) e condensado em **~365KB de documentação estruturada** em formato Markdown.

Esses documentos servem como a "memória de longo prazo" do sistema, alimentando os agentes de IA com conhecimento especializado sobre lançamentos, high-ticket, tráfego pago, copywriting, conteúdo e automação.

## Glossário de Termos

| Termo | Definição |
|-------|-----------|
| **6em7** | Faturar R$100.000 (6 dígitos) em 7 dias consecutivos, geralmente durante um lançamento. Meta inicial para infoprodutores no modelo Fórmula de Lançamento. |
| **Roma** | A "cidade" metafórica onde o público-alvo vive mentalmente. Representa o conjunto de dores, desejos, medos e aspirações do avatar. Toda comunicação deve "falar a língua de Roma". |
| **Avatar** | Representação semi-fictícia do cliente ideal. Inclui dados demográficos, psicográficos, dores, desejos, objeções e linguagem natural utilizada. |
| **CPL** | Conteúdo de Pré-Lançamento. Série de 3 vídeos estratégicos (CPL1, CPL2, CPL3) exibidos antes da abertura do carrinho para educar, gerar desejo e quebrar objeções. |
| **Faixa Preta / Faixa Marrom** | Níveis de maturidade do expert/lançador. Faixa Marrom = já fez pelo menos um 6em7. Faixa Preta = já fez múltiplos 6em7 ou 7em7 de forma consistente. |
| **SLG** | Sales Letter de Grupo. Apresentação de vendas feita para um grupo qualificado em contexto high-ticket. Combina elementos de pitch, webinar e sessão de vendas. |
| **1a1** | Sessão individual de vendas/diagnóstico. Conversa estratégica onde se aplica o framework de qualificação e fechamento para ofertas high-ticket. |
| **4 E's** | Framework para tratamento de objeções: **Escutar** (ouvir sem interromper), **Entender** (validar a preocupação), **Enquadrar** (recontextualizar), **Encaminhar** (direcionar para o fechamento). |
| **Andrômeda** | Metodologia de criativos para tráfego pago. Inclui 9 tipos de criativos, sistema de portfólio e regras de escala baseadas em performance. |
| **ASC** | Advantage Shopping Campaigns (ou Advantage+ no Meta Ads). Campanhas automatizadas pela IA do Meta que otimizam público, posicionamento e lances. |
| **SHT** | Super High-Ticket. Ofertas com ticket acima de R$10.000, geralmente mentorias, masterminds ou programas de acompanhamento individual. |
| **AR** | Audiência Recorrente. Pessoas que consomem conteúdo de forma consistente e repetida, formando a base de leads mais aquecida para lançamentos. |
| **Conteúdo Raiz** | Conteúdo profundo, de autoridade, com valor real. Oposto de "conteúdo Nutella" (superficial, genérico). O conteúdo raiz gera audiência qualificada e posiciona o expert como referência. |
| **Conteúdo Nutella** | Conteúdo superficial, raso, que não gera autoridade nem diferenciação. Conteúdo que qualquer um poderia produzir. Deve ser evitado como estratégia principal. |

## Benchmarks e Métricas de Referência

| Métrica | Valor de Referência | Contexto |
|---------|---------------------|----------|
| ROAS mínimo | 1.0 - 1.2 | Lançamentos (considerando LTV e recorrência) |
| CPL (Custo por Lead) | ~R$1,00 | Meta ideal para lançamentos via Meta Ads |
| Taxa de conversão carrinho | 7% - 10% | Conversão de leads em compradores no lançamento |
| Ciclos evolutivos | 7 ciclos | Número médio para atingir maturidade de lançamento |
| Escala diária de budget | 20% - 30% / dia | Aumento máximo recomendado de orçamento sem resetar aprendizado |
| Taxa de comparecimento live | 30% - 40% | Percentual de inscritos que assistem ao CPL ao vivo |
| Taxa de abertura email | 20% - 30% | Emails de lançamento para lista aquecida |
| Ticket médio high-ticket | R$3.000 - R$12.000 | Faixa de preço para ofertas high-ticket |
| NPS mentores | > 8.5 | Score mínimo para mentores delegados em operação high-ticket |
| CAC e-book isca | ~R$1,00 | Custo de aquisição usando e-book como isca digital |
| Faturamento framework e-book | R$12M | Case de referência usando framework de e-books + webinars |
