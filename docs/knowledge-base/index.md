# Base de Conhecimento IA-OS

## O que e a Base de Conhecimento

A Base de Conhecimento do IA-OS e um acervo de **28 documentos estruturados** organizados em **5 dominios** de marketing digital. Todo o material foi curado e sintetizado a partir de aproximadamente **~500MB de conteudo bruto** (videos, PDFs, lives, mentorias) e condensado em **~365KB de documentacao estruturada** em formato Markdown.

Esses documentos servem como a "memoria de longo prazo" do sistema, alimentando os agentes de IA com conhecimento especializado sobre lancamentos, high-ticket, trafego pago, copywriting, conteudo e automacao.

## Glossario de Termos

| Termo | Definicao |
|-------|-----------|
| **6em7** | Faturar R$100.000 (6 digitos) em 7 dias consecutivos, geralmente durante um lancamento. Meta inicial para infoprodutores no modelo Formula de Lancamento. |
| **Roma** | A "cidade" metaforica onde o publico-alvo vive mentalmente. Representa o conjunto de dores, desejos, medos e aspiracoes do avatar. Toda comunicacao deve "falar a lingua de Roma". |
| **Avatar** | Representacao semi-ficticia do cliente ideal. Inclui dados demograficos, psicograficos, dores, desejos, objecoes e linguagem natural utilizada. |
| **CPL** | Conteudo de Pre-Lancamento. Serie de 3 videos estrategicos (CPL1, CPL2, CPL3) exibidos antes da abertura do carrinho para educar, gerar desejo e quebrar objecoes. |
| **Faixa Preta / Faixa Marrom** | Niveis de maturidade do expert/lancador. Faixa Marrom = ja fez pelo menos um 6em7. Faixa Preta = ja fez multiplos 6em7 ou 7em7 de forma consistente. |
| **SLG** | Sales Letter de Grupo. Apresentacao de vendas feita para um grupo qualificado em contexto high-ticket. Combina elementos de pitch, webinar e sessao de vendas. |
| **1a1** | Sessao individual de vendas/diagnostico. Conversa estrategica onde se aplica o framework de qualificacao e fechamento para ofertas high-ticket. |
| **4 E's** | Framework para tratamento de objecoes: **Escutar** (ouvir sem interromper), **Entender** (validar a preocupacao), **Enquadrar** (recontextualizar), **Encaminhar** (direcionar para o fechamento). |
| **Andromeda** | Metodologia de criativos para trafego pago. Inclui 9 tipos de criativos, sistema de portfolio e regras de escala baseadas em performance. |
| **ASC** | Advantage Shopping Campaigns (ou Advantage+ no Meta Ads). Campanhas automatizadas pela IA do Meta que otimizam publico, posicionamento e lances. |
| **SHT** | Super High-Ticket. Ofertas com ticket acima de R$10.000, geralmente mentorias, masterminds ou programas de acompanhamento individual. |
| **AR** | Audiencia Recorrente. Pessoas que consomem conteudo de forma consistente e repetida, formando a base de leads mais aquecida para lancamentos. |
| **Conteudo Raiz** | Conteudo profundo, de autoridade, com valor real. Oposto de "conteudo Nutella" (superficial, generico). O conteudo raiz gera audiencia qualificada e posiciona o expert como referencia. |
| **Conteudo Nutella** | Conteudo superficial, raso, que nao gera autoridade nem diferenciacao. Conteudo que qualquer um poderia produzir. Deve ser evitado como estrategia principal. |

## Benchmarks e Metricas de Referencia

| Metrica | Valor de Referencia | Contexto |
|---------|---------------------|----------|
| ROAS minimo | 1.0 - 1.2 | Lancamentos (considerando LTV e recorrencia) |
| CPL (Custo por Lead) | ~R$1,00 | Meta ideal para lancamentos via Meta Ads |
| Taxa de conversao carrinho | 7% - 10% | Conversao de leads em compradores no lancamento |
| Ciclos evolutivos | 7 ciclos | Numero medio para atingir maturidade de lancamento |
| Escala diaria de budget | 20% - 30% / dia | Aumento maximo recomendado de orcamento sem resetar aprendizado |
| Taxa de comparecimento live | 30% - 40% | Percentual de inscritos que assistem ao CPL ao vivo |
| Taxa de abertura email | 20% - 30% | Emails de lancamento para lista aquecida |
| Ticket medio high-ticket | R$3.000 - R$12.000 | Faixa de preco para ofertas high-ticket |
| NPS mentores | > 8.5 | Score minimo para mentores delegados em operacao high-ticket |
| CAC e-book isca | ~R$1,00 | Custo de aquisicao usando e-book como isca digital |
| Faturamento framework e-book | R$12M | Case de referencia usando framework de e-books + webinars |
